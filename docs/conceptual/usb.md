|----------------------------------------|-----------|--------------------------|
| USB Type-A                             | 1996      | USB 1.0, 1.1, 2.0, 3.x   |
| USB Type-B                             | 1996      | USB 1.0, 1.1, 2.0        |
| Mini-A                                 | 2000      | USB 2.0                  |
| Mini-B                                 | 2000      | USB 1.1, 2.0             |

| Micro-B                                | 2007      | USB 2.0                  |
| Micro-AB                               | 2007      | USB 2.0                  |
| USB 3.0 Type-A (SuperSpeed Type-A)     | 2008      | USB 3.0, 3.1, 3.2        |
| USB 3.0 Type-B (SuperSpeed Type-B)     | 2008      | USB 3.0, 3.1, 3.2        |
